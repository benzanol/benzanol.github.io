Note: In order for gravity to appear most accurate, the total time should be around 20 seconds.

Each interval of the piecewise function has a time component (how long the interval should last for) and an equation component (the velocity of the train car during that time interval.)

This means that the velocity does not have to be continuous: It can suddenly speed up or reverse directions instantaneously.

Each equation should be a valid Javascript expression using the variable t for time. For example:

  • Normal Operations: t+1, 2*t, t/4
  • Exponents: t**2, 2**t, t**t
  • Trig: Math.sin(t), Math.cos(t), Math.tan(t)

The blue line shows velocity, the green line shows position, and the purple line shows the angle of the rod as a function of time.
